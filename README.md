## 初衷

- 完成个人记账系统服务端部分
- 转型成为全栈工程师的路上
- 签发jwt
- 验证jwt

## 主要架构

- @nestjs/typeorm
- @nestjs/jwt

- modules/auth
- modules/user

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```
